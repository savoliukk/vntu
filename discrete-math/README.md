# Discrete Mathematics

This directory contains artifacts for laboratory works 1-8 and practical works 1-3 in the discrete mathematics course.

## Structure

- `lab-01/` ... `lab-08/` - laboratory work reports, source scripts, results, and final diagrams.
- `practice-01/` - practical work 1 report and artifacts.
- `practice-02/` - practical work 2 report and artifacts.
- `practice-03/` - practical work 3 report and artifacts.
- `scripts/generate-reports.js` - regenerates the Markdown reports from computed JSON results.

Each laboratory directory has:

- `README.md` - final report.
- `*.py` - reproducible Python script.
- `*_results.txt` and diagrams/tables generated by the script.

Each practice directory has:

- `report.md` - final report.
- `artifacts/` - source scripts, rendered pages, extracted images, tables, and generated results.

Browser profiles and local cache folders are intentionally excluded.

## Run Locally

Run laboratory works:

```sh
python run_all.py
```

Run practical works:

```sh
npm run build
```

The practical work build script recomputes results and regenerates reports.

## Run With Docker

Run laboratory works:

```sh
docker compose up --build
```

Run practical works:

```sh
docker compose --profile practices run --rm practices
```
